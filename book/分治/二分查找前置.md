## 感想

如何确定二分的区间是不是包含m呢?

```
int m = (l+r)>>l;1
if( condition ){ //m绝对不可能答案
    bsearch(l,m-1);
}
else //m有可能是答案,也可能答案比m更大
    bsearch(m,r);
```
